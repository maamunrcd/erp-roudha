# Accessibility Standards

- Use semantic HTML elements and meaningful landmarks.
- Ensure keyboard navigation works for all interactive elements.
- Provide visible focus indicators.
- Use descriptive labels for form controls and buttons.
- Add alt text for informative images/icons when required.
- Use `aria-*` attributes only when semantic HTML is insufficient.
- Ensure loading and error states are announced/readable where appropriate.
- Maintain sufficient color contrast and readable text sizes.
- Do not rely on color alone to convey state.
